# Daily Helper
This is a apps of converting some data and scrapping images .

### Prerequisites
You will need PHP 7.0+ , <a href="https://yarnpkg.com/">yarn</a> on your machine


### Installing

<pre>
    git clone https://github.com/shah3093/daily_helper.git

    cd daily_helper

    yarn install

    cd backend/

    composer install

    cd ..

    yarn start

</pre>

## Built With

### For Frontend used
* React.js
* React Router Dom
* Semantic Ui React

### For Backend used
* PHP
* Guzzle
* Symfony Dom Crawler
* Symfony CSS  Selector
